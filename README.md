# Civic Guardian AI

An AI-powered civic safety platform designed to help citizens report, track, and resolve public issues efficiently in real-time.

---

## Overview

Civic Guardian AI is a smart web application that enables users to report civic problems such as potholes, garbage issues, water leakage, and more. The platform uses AI to prioritize complaints, ensuring faster and more effective resolution.

---

## Features

* 📍 Report civic issues with ease
* 🤖 AI-based issue prioritization
* 📊 Real-time tracking of complaints
* 🗺️ Location-based reporting
* 📈 Dashboard for monitoring issues
* 🔔 Status updates and notifications

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/civic-guardian-ai.git
```

2. Navigate to project folder:

```
cd civic-guardian-ai
```

3. Install dependencies:

```
npm install
```

4. Run the project:

```
npm run dev
```
